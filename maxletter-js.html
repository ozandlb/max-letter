Most Frequent Letter ("MFL")

Note:
Function is written in JavaScript.
Function returns an array. If a letter occurs with the same frequency as one or more other letters, all such letters are returned.
Function returns only letters, and ignores blank spaces, numbers and other non-letter characters.
Function makes no distinction between uppercase and lowercase letters.
Function efficiency is O(2N)
Example String
MFL
aaabbbcccde
####3333ff
GGGhhHH
